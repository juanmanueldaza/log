# Hello, World

This blog was sitting on an expired domain with lorem ipsum placeholders for too long. Time to fix that.

**log.daza.ar** is now the new home. No more ultravietnamita.com.ar — that era is done.

## Why a blog?

I build things — agent frameworks, tools, experiments. Most of it happens on GitHub, but the context, the detours, the _why_ behind the code never makes it into the commit messages. This is where that goes.

Expect posts about:

- NERV and agent architecture
- Hermes Agent workflows and tooling
- OSS experiments and failures
- The occasional 90s-retro-web appreciation post (this theme isn't going anywhere)

## Stack

This site is built with React + TypeScript + Vite, deployed via GitHub Pages. The 90s web aesthetic is intentional — monospace fonts, visitor counters, Netscape Navigator shoutouts. The dark/light toggle works too.

Posts are markdown files in `src/content/`. No CMS, no database, no tracking. Just files.

```
~/log.daza.ar $ ls src/content/
1.md
```

Let's see where this goes.
